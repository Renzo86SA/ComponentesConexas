Componentes Conexas

¿Que son las Componentes Conexas?

En Aritmética Modular, dos números enteros a y b son considerados congruentes con un modulo c si, al dividir ambos por n, dejan el mismo residuo.

Esto significa que a diferencia entre a y b es un multiplo de c, es decir, a - b = Kn para algún número entero k. La congruencia modular es clave en muchas ramas de las matemáticas y tiene aplicaciones importantes encriptografía, como el algoritmo RSA.

Calculo de Componentes Conexas

El inverso modular de un número a bajo un módulo n es el número x tal que el producto de a y x es congruente con 1 módulo n. Es decir, cumple la ecuación de la imagen.

Esto significa que x es el valor que, al multiplicarse por a, da un resultado que, al dividirlo por n, deja un residuo de 1. El inverso modular solo existe si a y n son coprimos (su máximo común divisor es 1).
